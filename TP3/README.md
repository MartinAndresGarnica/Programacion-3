# Clínica Salud+ Gestión de Turnos Médicos

Aplicación web desarrollada por **NovaTech** para la **Clínica Salud+**, destinada a la gestión de turnos médicos para pacientes.

---

## Instalación del proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/MartinAndresGarnica/Programacion-3.git
cd TP3
```

2. Instalar dependencias:

```bash
npm install
```

3. Ejecutar el proyecto:

```bash
npm run dev
```

El servidor iniciará en `http://localhost:3001`

---

## Tecnologías utilizadas

- Node.js
- Express
- EJS / Pug
- JWT (JSON Web Tokens)
